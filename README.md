# Supply Chain Analytics Platform

End-to-end analytics platform analyzing 180K+ orders to identify late-shipment risk, forecast demand, and surface margin leakage.

## Tech Stack
- **Database:** PostgreSQL (star schema)
- **Languages:** Python, SQL
- **ML:** scikit-learn, XGBoost, Prophet
- **BI:** Power BI / Streamlit
- **Tools:** Jupyter, Git, GitHub

## Project Structure
- `data/` — raw and processed datasets
- `sql/` — schema definitions and analytical queries
- `notebooks/` — EDA, modeling, forecasting
- `src/` — Python scripts (ETL, model training)
- `dashboards/` — BI dashboard files
